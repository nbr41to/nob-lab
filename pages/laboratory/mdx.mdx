# MDX

Markdown ファイルの中に JSX かけるようになる最強ライブラリ

[MDX](https://mdxjs.com/)

## サンプル

### ページをコンポーネントでラップ

`export default ({ children }) => <MdxBaseStyles>{children}</MdxBaseStyles>`

### Page をインラインにして HTML 要素のスタイルを細かく指定できる

[working with components](https://mdxjs.com/getting-started#working-with-components)

export default ({ children }) => <MdxBaseStyles>{children}</MdxBaseStyles>;
